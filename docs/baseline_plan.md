# 混交林树种分类 Baseline 实验方案

## 目标

用已有的开源方法（DUNIA/DeepForest），在我们的混交林数据上，定量回答一个问题：

> 这些在农业/纯林基准上验证过的 SOTA 方法，放到自然混交林里精度退化多少？

## 数据需求

一块林地的以下三类数据，时间相近（同一季节）：

| 数据 | 用途 |
|------|------|
| 无人机高分辨率影像（RGB 或多光谱） | 树冠纹理/光谱特征 |
| LiDAR 点云（机载） | 树高/三维结构 |
| 样地实测 | 树种标签（标准答案）、坐标、胸径 |

加分项：同一林地另一季节的重复飞行（用于物候对比），两块以上不同树种组成的林地。

## 实验 1：纯林 vs 混交林精度对比

**步骤**

1. 空间对齐：影像、点云、样地坐标统一到同一坐标系
2. 斑块划分：从样地数据标记出纯林斑块（单树种 > 80%）和混交林斑块
3. 分别跑 DUNIA 预训练编码器（或 DeepForest），各自输出树种分类精度
4. 对比两者差距

**预期产出**

纯林斑块 OA = X%，混交林斑块 OA = Y%，退化幅度 = X - Y

对标已有间接证据：Lee 2023 报告 CNN 纯林 F1 0.95 → 混交 0.75（-21%），Marinelli 2022 报告 ALS 纯林 OA 82-86% → 混交 64%（最差类），Quan 2023 报告中国次生林 OA 75.7%。

## 实验 2：单模态 vs 多模态

**步骤**

用 RGB 单独跑、LiDAR 单独跑、RGB+LiDAR 融合跑，同一套训练/测试划分。看融合能否缓解混交林退化。
