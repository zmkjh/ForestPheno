## 下载失败原因分析

1. **Cloudflare / 出版商反爬保护** — Nature(#5), Wiley(#13) 返回 "Just a moment..." 验证页
2. **ScienceDirect 浏览器检测** — Elsevier(#7) 拒绝非浏览器请求
3. **PNAS/Publishers 阻止 curl** — #16 返回HTML而非PDF
4. **机构库重定向到HTML** — Turin(#9), Cambridge/FEM(#13) 仅提供DSpace元数据页
5. **OSTI.gov 端口被封** — #18 443端口连接超时
6. **MDPI 返回 "Access Denied"** — MDPI主站反爬，但 CDN (`mdpi-res.com`) 可用

## arXiv 检索结论

18篇期刊论文均未以预印本形式发布在 arXiv。林学/遥感领域的核心期刊（RSE, Science, Nature Climate Change, PNAS）论文很少上传 arXiv 预印本。

## GitHub 检索结论

未发现包含这些论文 PDF 副本的仓库。找到的仓库均为代码实现（如 `tree-detection-evo`, `TreeSpeciesClassification`），不含论文PDF。`awesome-forests` GitHub topic 为空。
